# 🔐 Responsive Login & Registration UI

## 📌 Project Overview
This project is a responsive Login and Registration user interface developed using HTML, CSS, Bootstrap, and JavaScript. It includes form validation, password security features, and a clean user-friendly design.

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome

---

## ✨ Features
- Responsive design (mobile & desktop)
- Login and Registration forms
- Strong password validation
- Show/Hide password toggle
- Inline error and success messages
- "Remember Me" option
- "Forgot Password" link
- Dummy email check (AJAX simulation)

---

## 🔒 Password Requirements
- Minimum 8 characters
- At least 1 uppercase letter
- At least 1 lowercase letter
- At least 1 number
- At least 1 special character

## 🚀 How to Run
1. Download or clone the repository
2. Open `index.html` in any browser
3. Navigate between Login and Register pages
